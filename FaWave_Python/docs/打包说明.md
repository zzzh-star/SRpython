# 打包说明

打包生成的可执行文件深度依赖于程序资源目录中的配置文件与 3D 渲染所需的数据资产，因此运行完 `build_exe.bat` 后，您需要**复制整个 `dist/FaWave` 文件夹**以迁移软件。
* `build_exe.bat` 脚本内会使用 `--add-data "assets;assets"` 命令选项来确保 `assets/models` 下的模型（如 `assembly_model.glb`）被打包入内。
* 打包后 Python 的运行时环境已静态链接 `requirements.txt` 中指定的依赖：`PyOpenGL`、`pyqtgraph` 与 `trimesh`。
* 任何情况下都**不要仅提取 exe 单文件拷贝**，否则会导致软件缺失图标、样式并出现无法加载模型等异常回退现象。
