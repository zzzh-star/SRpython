# 三维模型说明

真实模型默认加载路径为 `assets/models/assembly_model.glb`，程序同样支持 GLB / GLTF / OBJ / STL 等文件格式进行加载。
* 推荐使用 GLB 格式以展示带颜色的彩色装配体。
* 文件名必须使用纯英文，请避免在路径中使用中文名称或特殊字符。
* 此模型通过在 3D 视图中与用户提供基于鼠标和滚轮的视角交互：
  * **鼠标滚轮**：按鼠标所在位置进行缩放；
  * **按住鼠标滚轮并拖动**：旋转三维模型；
  * **鼠标右键按住拖动**：平移三维模型；
  * **鼠标左键双击**：重置当前模型的显示视角。
* 若依赖缺失、资源缺失或 OpenGL 在当前平台不可兼容，软件均会展示基于代码生成的简化占位三维集合视图 (Fallback Geometry)，且不影响系统数据采集、保存等核心通讯功能。
## 图片预览
模型卡片默认优先显示一张静态预览图 `assets/images/model_preview.png` 以节约运行资源。只有当用户主动点击卡片上方的 **“查看 3D 模型”** 后，才会对上述几何图形或 `assembly_model.glb` 资源进行实例化加载。若图片资源缺失，UI 也会提供适当的边界框和提示反馈。
