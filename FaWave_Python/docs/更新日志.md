# 更新日志

## 2026-05-27
* 新增 3D 模型显示支持，可加载 `assets/models/device_model.stl` 并在 UI 中实时动态渲染。
* 添加 "牵拉/剪切" 双任务模式，支持相应的异常状态监控预警。
* 更新背景异步保存记录，新增相关报警元数据列 (`TaskMode`, `AlarmEvent`, 等)。

## 2026-05-28
* 修复 3D 模型加载逻辑，修复 SyntaxError 错误，添加真实的 GitHub 仓库 STL 文件至 `device_model.stl`。
* 移除全部硬编码占位符报警文本，真实绑定根据模式状态实时动态渲染异常报警卡片（"组织撕裂"，"牵拉过载" 等）。
* 重构所有说明文档为纯中文。

## 2026-05-28 (v2)
* 全面接入并在 UI 中融合了外部 Codex 分支所完成的 3D 模型交互与加载程序 (`model_loader.py` & `model_viewer.py`)，目前支持 `trimesh` 库对多格式 (GLB/GLTF/OBJ/STL) 的解析与渲染。
* 模型系统默认挂载 `assembly_model.glb` 作为主力展示彩色装配体。
* 根据用户视觉反馈对 MainWindow 的总体 UI 比例进行了宽度的精细微调，使模型卡片保持约占全宽 25%，并保障原数据观测及波形曲线等主要信息的平铺排版空间更加充分。
